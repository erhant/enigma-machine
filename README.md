# Enigma Machine
A small enigma machine for COMP543 - Modern Cryptography course. 

For the proof of concept, two users will talk to each other via a file. The file stored encrypted messages, which the users decrypt as they recieve. An eavesdropper can only see the file, and with no access to the rotor configuration, it can only brute-force to find the content of the message. See the report for more details and instructions.
